# TodoMVC with Vue Composition API

> This app is written in Vue 2 and Composition API

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```

```
